### 3. **Use Supabase “Table Editor” UI**

* Go to [app.supabase.com](https://app.supabase.com)
* Select your project ➝ click **Table Editor**
* Add new rows manually to each table.

---

### 4. **Use Supabase CLI + SQL Scripts (Advanced)**

If you want reproducible local data:

```bash
supabase db reset
supabase db seed < seed.sql
```

Where `seed.sql` contains your dummy inserts.

---

### 5. **Generate Fake Data with Faker + Drizzle ORM (Programmatic)**

Install dependencies:

```bash
npm install @faker-js/faker drizzle-orm
```

Example script (`scripts/seed.ts`):

```ts
import { db, users } from '../server/db';
import { faker } from '@faker-js/faker';
import { eq } from 'drizzle-orm';

async function seedUsers() {
  const fakeUsers = Array.from({ length: 10 }).map(() => ({
    id: faker.string.uuid(),
    name: faker.person.fullName(),
    email: faker.internet.email(),
    created_at: new Date(),
  }));
  await db.insert(users).values(fakeUsers);
}

seedUsers();
```

Run:

```bash
tsx scripts/seed.ts
```
